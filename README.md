# MakeMeet Telegram Bot

Telegram-интерфейс для админ-аналитики MakeMeet.

## Что делает
- Получает данные через Gateway
- Показывает:
  - 📊 Статистику
  - 🆕 Посты
  - 👥 Пользователей
  - 🔥 Спринты
- Даёт кнопку "Открыть в MakeMeet"

## Запуск

```bash
cd makemeet-tg-bot
npm install
node index.js
